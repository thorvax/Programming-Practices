WEB DEVELOPMENT

What is Web Development?

Web development is the process of creating websites and web applications. It involves a combination of programming, design, and content management to build functional and visually appealing sites that can be accessed via the internet.

What languages are used in Web Development?

HTML

Below is an example of a simple HTML document structure. 

            
                
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My Web Page</title>
        <link rel="stylesheet" href="styles.css">
    </head>
    <body>
        <header>
        </header>
        <main>
        </main>
        <footer>
        </footer>
    </body>
</html>
